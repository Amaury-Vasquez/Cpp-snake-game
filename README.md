## Requirements

- Linux-based OS
- g++ compiler
- ncurses library installed [install ncurses] (https://ostechnix.com/how-to-install-ncurses-library-in-linux/)

## Compile

```bash
g++ main.cpp game.cpp snake.cpp food.cpp -o main -lncurses
```

## Usage

```bash
./main
```
